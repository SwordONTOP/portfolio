Robot Suiveur de Ligne TERMINALE

Un robot autonome qui suit une ligne noire grâce à des capteurs infrarouges. Projet réalisé en équipe avec une interface de contrôle Python.

Arduino Nocode Python Électronique

Description

Le robot suiveur de ligne est un projet qui combine les compétences en d'Assemblage et un peu d'electronique et en programmation. Il utilise un microcontrôleur Arduino et des capteurs infrarouges pour suivre une ligne noire sur un fond blanc. Le robot est controler par une application faite par mo groupe et moi .

Compétences

  • Programmation Arduino/C++
  • Électronique embarquée
  • Interface Arduino-Python
  • Assemblage des pieces
  • Contrôle desktop capteurs

Objectifs

  • Créer un robot autonome capable de suivre une ligne
  • Optimiser la détection de ligne avec des capteurs infrarouges

Techniques et Savoir-faire Acquis

  • Maîtrise de l'interface Arduino-PC via Serial
  • Programmation orientée événement pour la détection de ligne
  • Calibration de capteurs infrarouges
  • Optimisation de la vitesse et de la précision du robot
  • Travail collaboratif et gestion de projet

Application Smartphone de Contrôle Robot (Bluetooth) TERMINALE

Application Android permettant de contrôler un robot via Bluetooth. Interface utilisateur intuitive avec contrôle de direction et vitesse.

Android MIT App Inventor Bluetooth

Description

L'application de contrôle Bluetooth est une application Android qui permet de contrôler un robot à distance via une connexion Bluetooth. L'interface utilisateur intuitive permet de piloter le robot avec des commandes de direction (avancer, reculer, tourner) et de vitesse.

Compétences

  • Développement Android
  • Programmation bloc avec MIT App Inventor
  • Mise en place de la com Bluetooth

Objectifs

  • Développer une interface utilisateur intuitive
  • Implémenter la communication Bluetooth stable
  • Créer un système de contrôle précis
  • Optimiser la consommation d'énergie

Techniques et Savoir-faire Acquis

  • Développement d'applications Android
  • Communication Bluetooth
  • Design d'interface utilisateur mobile
  • Gestion des erreurs de connexion

Jeux Snake Premiere

Une version moderne du jeu Snake classique, implémentée avec HTML5 Canvas et JavaScript. Le serpent se déplace sur une grille et doit manger des pommes pour grandir tout en évitant les collisions avec les murs et lui-même.

Python Animation Jeux

Description

Une implémentation moderne du jeu Snake classique. Le jeu utilise des bibliothèques de jeu Python (Pygame) pour l'affichage graphique et la logique de jeu. Le serpent se déplace à travers une grille, mange des pommes pour grandir et doit éviter les collisions avec les murs et lui-même.

Compétences

  • Développement avec Python
  • Animation avec Pygame
  • Gestion des événements clavier
  • Logique de collision

Objectifs

  • Créer une version moderne du jeu Snake
  • Implémenter un système de score
  • Ajouter des effets visuels avec Pygame

Techniques et Savoir-faire Acquis

  • Développement de jeux
  • Animation
  • Gestion des événements du jeu
